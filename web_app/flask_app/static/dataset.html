Dataset i processament d'imatges

El model va ser entrenat utilitzant el dataset COVIDx CXR-2 disponible de manera gratuïta.

Resumen

Aquest dataset està compost per 29.986 imatges etiquetadesde radiografies toràciques, de les quals 13.992 són radiografies negatives per a COVID-19 iles restants 15.994 imatges són radiografies positives per a COVID-19.

Processament del dataset

El dataset, una vegada descarregat i descomprimit, compta amb dues carpetes amb imatges d'entrenament i proves, addicionalment té dos arxius de text amb les metadades de cadascuna de les imatges de cada carpeta.

Primer es realitza un escaneig per a cada carpeta del dataset i es crea una llista amb el camí (path) de cadascuna de les imatges. 

          
            def scan_covidx_dataset(dataset_path, partition = 'train'):

Processament de les imatges

Després de tenir la llista amb les imatges, aquestes són importades amb la llibreria opencv i processades, primerament normalitzades i després aplicant un filtratge CLAHE, basat en l'article "IMAGE ENHANCEMENT ON DIGITAL X-RAY IMAGES USING N-CLAHE" dels autors (Koonsanit et al) disponible en el següent enllaç. Per a finalment escalar la imatge a 400x400.
